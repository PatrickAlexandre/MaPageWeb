# 🛠️ Pile Technologique
## Bibliothèques et frameworks JavaScript
- **Bibliothèque** : Collection de fonctions que vous pouvez appeler pour réaliser des tâches spécifiques.
- **Framework** : Cadre structurel qui guide la manière dont vous construisez et organisez votre application.
    - Frontend : React, Vue.js
- **API** : Interface permettant la communication et l'interaction entre différentes applications ou services.

## Frontend
- 🌐 **Vue.js** : Un framework JavaScript progressif pour la création d'interfaces utilisateur.
- 🎨 **Bootstrap** : Un framework CSS populaire pour développer des sites web réactifs et mobiles en priorité.
- ⚡ **jQuery** : Une bibliothèque JavaScript rapide, petite et riche en fonctionnalités pour la manipulation du DOM et la gestion des événements.

## Visualisation de données
- 📊 **Google Charts** : Une bibliothèque puissante, simple à utiliser et gratuite pour créer des graphiques et des diagrammes, facilitant ainsi la visualisation des données.
- 🖼️ **Font Awesome** : Un kit d'outils pour les icônes et les logos sociaux afin d'ajouter des éléments visuels à l'application.
